How to use:

By default, all categories listed under Category Unlocks are hidden. You can review that list to see if you are interested in unlocking any of those categories

  • How to Unlock Categories: If you click on any circle BUT Limit (dark red) and Not Entered (white), it will unlock a category with more items on the specified topic
  • Custom Items: If you want to add your own custom item to the list, click the gear symbol in the top left next to Obnoxiously Detailed Kinklist. This will open up a text file you can edit, and you just follow the formatting specified, using * lines to add a new item, ? lines to add an explanation for that item, and if you want new categories, they are defined with # then below that you define how many columns it will have and what the names of those columns will be (Column 1, Column 2)

If an item does not apply to you, you can just mark it as Not Entered, the white color. This can be for any reason you like, you only answer what you want to

Once finished with filling out the form to your liking, you can press the Download button
