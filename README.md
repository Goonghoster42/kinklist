# Kinklist

An interactive web-based kinklist tool for creating and sharing preferences.

## About

This is a fork of [adhesivecheese's kinklist](https://github.com/adhesivecheese/kinklist) with improvements and bug fixes by saustinlabs, then edited by Goonghoster42 to have many many extensive categories and options.
